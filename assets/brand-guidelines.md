# HUDForge Brand Guidelines

## Positioning
HUDForge is a premium Roblox UI workflow platform. It is not an AI art toy. It helps creators describe a UI, preview it, refine it, and export direction they can actually use in Roblox Studio.

## Brand promise
- **Core phrase:** Describe a UI. Preview it. Refine it. Export to Roblox.
- **Motto:** Build Your World.
- **Audience:** Roblox solo devs, small teams, game jam builders, creators with scripting skill but weak UI design confidence.

## Visual system
### Primary logo files
- `/public/brand/hudforge-wordmark.svg`
- `/public/brand/hudforge-emblem.svg`
- `/public/brand/hudforge-stackmark.svg`
- `/public/brand/social-avatar.png`

### Logo usage
- Use the **wordmark** on website headers, presentations, press PDFs, and social cover overlays.
- Use the **emblem** for app icons, favicons, Discord server icons, and square avatars.
- Maintain at least 24px clear space around all logo variants.
- Avoid stretching, recolouring outside the approved palette, or placing the logo on busy mid-tone backgrounds without a dark scrim.

### Colour palette
| Token | Hex | Use |
|---|---:|---|
| Void | `#050816` | Page background |
| Forge Panel | `#0B1020` | Cards, nav, overlays |
| Cyan Core | `#00D1FF` | Primary CTA, highlights |
| Cyan Glow | `#52E3FF` | Secondary glow, hover accent |
| Arcane Violet | `#7C5CFF` | Secondary accent, premium depth |
| Ember Gold | `#FFBB33` | Rare highlight, badges |
| Ice White | `#F8FBFF` | Primary copy |
| Slate Mist | `#94A3B8` | Secondary copy |

### Typography
- **Display:** Space Grotesk for headlines and hero copy.
- **Body:** Inter for interface copy and supporting text.
- **Mono/system notes:** Geist Mono for labels, metrics, and UI telemetry.

## Imagery direction
- Premium cyber-fantasy.
- Dark navy scenes with neon cyan and violet energy.
- Builder/hero characters, not generic stock photos.
- UI panels should feel Roblox-native and production-ready.
- Prefer compositions with negative space for copy overlays.

## Voice and tone
- Sharp, builder-focused, and grounded.
- Confident without hype.
- Specific about workflows, exports, templates, and time saved.
- Never sound like a generic “create anything with AI” startup.

## Messaging pillars
1. **Roblox-native workflow** — built for inventories, quest screens, shops, HUDs.
2. **Faster first usable result** — templates and guided prompts reduce blank-page paralysis.
3. **Export-aware outputs** — transparent assets, prompt context, and Luau hierarchy notes.
4. **Premium aesthetic** — output should look game-ready, not placeholder UI.

## Social channel look
- Use the avatar emblem or `social-avatar.png` for square profile images.
- Use generated headers in `/public/generated/brand/`.
- Keep bios short, builder-first, and conversion-oriented.

## Accessibility rules
- Keep text/background contrast AA or better.
- Do not place cyan text directly on bright hero areas without a scrim.
- Use gold only for emphasis, not body copy.
- Minimum interactive target: 44x44px.
