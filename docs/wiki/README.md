# HUDForge Local Wiki

Simple, plain-English notes about how HUDForge works and what changed.

This wiki is for Luke and Hermes. It should be updated whenever meaningful product, database, auth, billing, generation, export, deployment, or infrastructure changes are made.

## Rule

When changing HUDForge, also update this wiki with:

1. What changed
2. Why it matters
3. How to verify it
4. Any blocker or follow-up

Keep it simple. Avoid jargon unless it is necessary.
