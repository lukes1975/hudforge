# HUDForge Distribution Strategy

## Why this matters
HUDForge is still pre-proof. Distribution has one job right now: drive qualified Roblox creators into the first useful workflow and turn that into feedback, proof, and conversions.

## Primary goal for the next 7 days
- Increase qualified early-access signups
- Capture which UI categories creators actually want
- Start a community loop that improves activation

## Channel priorities
### Tier 1 — do now
- X / Twitter clips
- Discord waitlist-to-feedback loop
- Site CTA + email capture
- Roblox DevForum post

### Tier 2 — support proof
- YouTube Shorts
- LinkedIn credibility posts
- GitHub README and launch notes

### Tier 3 — after initial traction
- Long-form YouTube breakdowns
- Press outreach
- Newsletter cadence beyond launch updates

## Content categories
1. AI builds a Roblox shop UI in seconds
2. Bad UI -> premium HUD redesign
3. Inventory prompt breakdowns
4. Quest / reward screen generation
5. Roblox Studio handoff demo
6. Founder build-in-public snippets

## Operating cadence
- 3 short clips/posts per day during launch week
- 1 email update when meaningful product changes ship
- 2–3 direct founder outreach messages per day to relevant Roblox creators or micro-studios

## Funnel design
1. Short clip or launch post
2. Click to site
3. Waitlist signup
4. Welcome email with template gallery and examples
5. Invite top responders into Discord or beta list
6. Capture their requested UI type and friction points

## Best CTA language
- Join the founding creator queue
- See the template arsenal
- Get early access to premium Roblox UI workflows
- Show me the current HUD templates

## KPI targets
- Landing page conversion > 8%
- Waitlist signup cost as low as possible through organic content first
- At least 10 high-signal feedback replies from creators
- At least 3 repeated requests for the same UI type before building deeper features
